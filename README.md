# 🏛️ Audit-Grades Pro: Sistema de Gestión Académica Inmutable

> *"Non scholae sed vitae discimus"*

## 📝 Descripción del Proyecto
**Audit-Grades Pro** es una solución de ingeniería de software diseñada para la carga y gestión de calificaciones académicas dentro de la UNEXCA. El sistema pone un énfasis crítico en la integridad de los datos mediante un **Log de Auditoría Inmutable**, asegurando que cualquier alteración en los registros académicos deje una traza permanente y auditable.

## 🛡️ Características Principales (Catedral Digital)
- **Persistencia Blindada:** Base de datos relacional robusta bajo el estándar ACID utilizando SQLite.
- **Auditoría Nativa:** Implementación de disparadores (Triggers) a nivel de base de datos para garantizar la trazabilidad de cada cambio.
- **Escudo de Excepciones:** Validación estricta de entradas para prevenir errores de usuario y corrupción de datos.
- **Interfaz Profesional:** Experiencia de usuario moderna desarrollada con la librería `CustomTkinter` en modo oscuro.

## ⚙️ Requerimientos Técnicos
- **Lenguaje:** Python 3.10+
- **Base de Datos:** SQLite 3
- **GUI:** CustomTkinter
- **Metodología:** Scrum (Sprints 5-10)
- **Versionamiento:** Git / GitHub Flow
